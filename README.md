## Nível 1: Iniciante
O Sistema de numeração Binário é o sistema que todos os computadores digitais se baseiam. Portanto é importante que os desenvolvedores entendam a matemática binária ou de base 2. O objetivo do Bin2Dec é fornecer a prática e entendimento dos cálculos binários.

O Bin2Dec permite que o usuário insira sequências de até 8 dígitos binários, 0 e 1, em qualquer ordem e, em seguida, exibe seu equivalente decimal.

Esse desafio exige que o desenvolvedor que o implemente, siga as seguintes restrições:

* Vetores/Matrizes não podem ser usados para conter os dígitos binários inseridos pelo usuário.
    
* A determinação do equivalente decimal de um dígito binário específico na sequência deve ser calculado usando uma única função matemática, por exemplo, o logaritmo natural. Cabe a você descobrir qual função usar.

## Histórias do Usuário

- [x] O usuário pode inserir até 8 dígitos binários em um campo de entrada.
- [x] O usuário deve ser notificado se algo diferente de 0 ou 1 foi inserido.
- [x] O usuário visualiza os resultados em um único campo de saída contendo o equivalente decimal (base 10) do número binário que foi inserido.

## Recursos Bônus

- [X] O usuário pode inserir um número variável de dígitos binários.
- [x] O usuário não pode inserir nada além de 0 ou 1.
