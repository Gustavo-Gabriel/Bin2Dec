## Nível 1: Iniciante
O Sistema de numeração Binário é o sistema que todos os computadores digitais se baseiam. Portanto é importante que os desenvolvedores entendam a matemática binária ou de base 2. O objetivo do Bin2Dec é fornecer a prática e entendimento dos cálculos binários.

O Bin2Dec permite que o usuário insira sequências de até 8 dígitos binários, 0 e 1, em qualquer ordem e, em seguida, exibe seu equivalente decimal.
